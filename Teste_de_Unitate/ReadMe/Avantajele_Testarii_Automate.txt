

	Avantajele 'Testarii Automate'




-------------------------------------------------------------------------------------------------------------

   (1) Testarea mai Frecventa a Codului , în mai Puțin Timp;
   (2) Prinderea Erorilor înainte de Implementarea Aplicatiei;
   (3) Permite Implementarea Aplicației cu Mai Multă Incredere;
   (4) Permite Refabricarea/Restructurarea Codului cu Incredere;
   (5) Ajută la Concentrarea Mai Mare pe Calitatea Aplicatiei.

-------------------------------------------------------------------------------------------------------------




  >> "Refabricarea"
	=> inseamna 'Schimbarea  Structurii Codului'
	=> pentru al face cat mai 'Curat'  si 'Mentenabil',
	=> fara a ii 'Schimba Comportamentul / Functionalitatea'.



  >> Exemple de 'Refabricare':
	=> Extragerea unor Linii de Cod dintr-o Metoda
		-> intr-o Metoda Privata Separata.
	=> Redenumirea unei Metode
